UML
1. 정의 : 다양한 수준의 개발자가 참가하여 진행되는 프로젝트는 복잡한 시스템(하드웨어, 소프트웨어의 구현, 커뮤니케이션, 이윤관계)과
            시시각각 변화하는 요구사항에 기반하며, 프로젝트 진행 간 도출될 수 있는 다양한 문제점의 예측 관리는 복잡도를 단순화하는것으로부터
            시작된다. 복잡도를 단순하게 시각화하고 리스트 예측 관리를위한 용도로 활용될 수 있는 UML[Unified Modeling Language]은
            보다 견고하고, 안정적이고, 유동적인 프로젝트를 진행하는 활용된다.
   
2. 프로젝트 수행 간 현상.
    매년 200,000여개의 프로젝트가 수행되고 3조원 정도가 소요됨.
    해당 기간에 계획된 모든기능을 구현되는 프로젝트 성공율은 16%에불과함.
   31%는 품질 문제로 취소됨.
   53%는 예산을 평균 200% 초과함.
    수행된 프로젝트에서 계획된 기능의 평균 50%만이 구현됨.
    
3. 프로젝트 실패의 원인
   3.1 비현실적인 일정 계획 수립
   3.2 프로젝트 관리의 비효율
   3.3 비효율적이고 잘못된 설계와 빈번한 요구사항 변경 -> UML로 극복 가능
   3.4 개발자 능력부족
   3.5 개발자의 지식부족
   3.6 개발자의 처우 빈약   
   
4. UML의 활용
   4.1 복잡한 프로세스의 단순한 시각화
   4.2 전체 프로젝트를 관망할 수 있는 다양한 관점의 이해 및 분석
   4.3 리스크 예측 관리 용이
   4.4 다수의 이해당사자 간 커뮤니케이션의 기초자료로 활용
   4.5 UML로부터 실제 코드 원형의 생성 가능

6. AmaterasUML Plug-in 설치
   6.1 이클립스에 eGovFramework AmaterasERD 또는 eGovFramework AmaterasUML plug-in이 기
           설치되어 있으면 uninstall하고 진행 
   
       http://amateras.sourceforge.jp/cgi-bin/fswiki_en/wiki.cgi?page=AmaterasUML
          에서 1. Please download AmaterasUML_x.x.x.zip from here. 의 here 클릭 
       AmaterasUML_1.3.4.zip 다운로드
       
   6.2 압축을 푼 폴더 및 파일을 사용중인 이클립스 폴더의 plugins 폴더로 이동
   6.3 이클립스 reboot 
   6.4 생성마법사 -> amaterasUML의 특정 UML 메뉴 선택 -> 저장
     
     * 생성마법사 -> amaterasUML 메뉴가 출력되지 않을때
         Help -> Install new software -> 
         https://hudson.eclipse.org/hudson/job/gef-master/lastSuccessfulBuild/artifact/update-site/ 설치
         
   6.5 UML 작성 에디터의 그리드 출력
       window -> preferences -> AmaterasUML -> show grid 선택 적용
         
7. UML 모델링 준비
   7.1 개발자 입장에서 자동차를 추상화(중요하고, 필수적인 사항에 집중하여, 공통적인 사항을 도출) 하도록 함
       
       7.1.1 자동차 특성 : 모든 자동차는 차체가 존재함
                                  엔진을 포함함
                                  운전대가 존재함
                                  변속기가 존재함
                                  바퀴가 존재함
                                  헤드라이트가 존재함
                                  엑셀레이터가 존재함
                                  브레이크가 존재함
                   자동차 행위 : 시동을 건다
                                  시동을 멈춘다
                                  가속한다
                                  감속한다
                                  좌회전한다
                                  우회전한다
                                  후진한다
