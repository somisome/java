p를 클릭할 때 마다 폰트사이즈가 5px씩 증가한다.
글자색상도 매번 랜덤으로 변경한다.

무궁화 꽃이 피었습니다.





p - 폰트컬러 랜덤변경, 폰트사이즈 증가

$('p').on('click',function(){
    a = Math.floor(Math.random()*255);
    b = Math.floor(Math.random()*255);
    c = Math.floor(Math.random()*255);
    $(this).css({'font-size':'+=5px','color':'rgb('+a+','+b+','+c+')'});
})

이벤트 해제

$('button').on('click',function(){
    $('p').off('click');
})
