$('.bb').bind('click',function(){
    $('<button/>',{'text':'버튼'+count}).appendTo('#d2');
})

$('#but').bind('click',function(){
    $('.bb').unbind();
})

bind()메소드

선택자로 검색한 모든 엘리먼트에 핸들러를 등록
처음부터 body안에 만들어져 있었던것에만 이벤트가 적용
새롭게 만들어지는 것에는 이벤트가 적용되지 않는다.

delegate(선택자, 이벤트명, 핸들러)

미래에 추가될 엘리먼트에 대해서도 핸들러를 미리 등록하는 메서드
처음부터 body안에 만들어져 있었던 것과 새롭게 만들어지는 것 모두 이벤트 적용
$(document).delegate(선택자, 이벤트명, 핸들러 )

모든 이벤트 등록 메소드 - 통합 메소드 (1.7버전에서 도입)

on(이벤트명, [선택자], 핸들러)
off( 이벤트명, [선택자], 핸들러)
>> 선택자 가 있으면 delegate와 같고 없으면 bind와 같음
