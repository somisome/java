1.jQuery라이브러리 사용하는 방법 2
 script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"
 웹 상의 경로를 사용하여 js파일을 부른다.

2.method chaining
 var pp = $('p');
 pp.css('color','pink');
위의 두 줄을 한줄로 표현가능
 -> $('p').css('color','pink');

3.p태그가 걸려있는 문장 중 특정 문장만 스타일을 바꾸고 싶을 때
 $('p:eq(3)').css('color','forestgreen');
 처럼 $('p:eq(3)')로 표현한다. (3)이 인덱스를 특정지어주는 파트

javascript에서는 다음과 같이 표현함
 pp = document.getElementsByTagName('p');
 pp[1].style.color = 'lightpink';
 pp[2].style.color = 'pink';
 pp[3].style.color = 'hotpink';
 pp[4].style.color = 'tomato';
