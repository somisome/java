피카츄 라이츄 파이리 꼬부기 버터플

get( ) , get( N ) , [N]

jQuery 객체에 대해 dom 메소드 호출 해야 하는 경우
get()함수는 매치되는 DOM요소들을 배열로 반환
get(N)함수는 DOM요소 배열에서 N번째 요소 반환([N])
N이 배열 범위를 넘을 경우 “undefined”를 반환
배열의 첨자는 0부터 시작됨

빈객체 비교 할 경우 : if($('h1').length == 0)

예 1:

$('p').css('color','blue');
var plist = $('p');

- jquery객체로 부터 dom객체를 얻음
var dom = plist.get(2);

- dom객체는 css()호출할 수 없음
//dom.css('font-size','2.0em'); ←사용불가
dom.style.color='red'; ←사용가능

- dom객체를 $()로 감싸면 jquery객체로 변환
$(dom).css('font-size','2.0em'); ←사용가능

- 1번째 dom객체 반환
dom = $('p')[1];
dom.style.background='linen';
//dom.css('background','linen'); ←사용불가
$(dom).css('border','1px solid pink'); ←사용가능

예 2:

- span 전체적으로 테두리 주기
$('span').css('border','1px solid forestgreen');

- 두번째 span만 background linen으로. jquery객체로 변환.
$($('span')[1]).css('background','linen');

- 세번째 span만 글씨색 바꿀 때. dom객체 style사용
$('span')[2].style.color='tomato';

- 네번째 span만 background pink로. eq(n) 사용.
$('span:eq(3)').css('background','pink');
