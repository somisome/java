DOM - JQuery

jquery:
var jq_h1=$('h1');
script:
var dom_h1=document.getElementsByTagName("h1")[0];
$(jq_h1).css('color','sienna');

dom객체로 css메소드를 호출불가

-> 오류남 -> dom_h1.css('color','forestgreen');
-> 오류안남 -> dom_h1.style.color='forestgreen';

dom객체를 jquery객체로 변환

$(dom_h1).css('font-size','5.0em');

jquery객체를 dom객체로 변환(get함수)

jquery객체 jq_h1:
   var jq_h1=$('h1');
jq_h1을 dom객체로 변환한 jqdom:
   var jqdom = jq_h1.get(2);
